# 五子棋游戏设计文档

## 项目概述

基于 HTML Canvas 的五子棋双人对战游戏，两个玩家在同一设备上轮流落子，自动判断胜负。

## 功能需求

### 必须功能
- [x] 15×15 标准五子棋棋盘
- [x] 使用 HTML Canvas 绘制
- [x] 黑方先手，轮流落子
- [x] 点击棋盘空白处落子
- [x] 自动判断胜负（连成五子即获胜）
- [x] 显示获胜方提示
- [x] 重新开始游戏

### 排除功能（保持简单）
- 无 AI 对手
- 无悔棋功能
- 无计分/历史记录
- 无网络对战

## 技术架构

### 文件结构
```
├── index.html      # 入口文件，包含画布和基本结构
├── style.css       # 样式，居中布局，美化界面
└── app.js          # 游戏逻辑
```

### 不需要
- 无构建工具
- 无 npm 依赖
- 直接浏览器打开即可运行

## 核心算法

### 胜负判断
落子后检查该位置四个方向（横向、纵向、左上-右下、右上-左下）是否有连续五个同色棋子。

### 坐标转换
将鼠标点击的像素坐标转换为棋盘网格坐标。

## UI 布局
- 棋盘居中显示
- 上方或下方显示当前回合/获胜提示
- 提供重新开始按钮
- 简洁现代的浅色风格

## 成功标准
- 正常落子，切换玩家
- 正确判断胜负
- 点击重新开始可以清空棋盘重新游戏
- 在现代浏览器中能正常运行
